# 想吃便利贴墙

一个本地优先的美食探店记录网站，用来保存从小红书看到的想打卡店铺。第一版采用“链接 + 帖子正文辅助粘贴”的方式，用规则解析生成可编辑预览，再保存到按添加日期分组的便利贴墙。

## 功能

- 粘贴小红书链接和帖子正文，自动提取店铺名称、类型、位置、人均、推荐菜等字段
- 保存前可编辑解析结果
- 按添加日期生成便利贴分组
- 按店铺类型和自定义标签筛选
- 支持待打卡、已打卡状态
- 已打卡记录可补充日期、评分和备注
- 使用浏览器本地存储保存数据

## 本地运行

```bash
npm install
npm run dev
```

访问 `http://127.0.0.1:3000`。

## 常用命令

```bash
npm run lint
npm run build
```
